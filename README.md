# 🎮 Games Hub

A fun and interactive collection of browser-based mini games built for entertainment, speed, and learning.  
This project includes multiple games inside a single platform with a clean UI and smooth gameplay experience.



https://github.com/user-attachments/assets/1d712258-4e95-4331-8ea1-09ce60b4a3bb



## 🚀 Features

- 🎯 Multiple mini games in one platform
- 🧠 Memory and puzzle-based gameplay
- ⌨️ Typing speed challenge
- 🔢 Number guessing game
- 🔤 Word scramble game
- 📱 Responsive design
- ⚡ Fast and lightweight interface

---

## 🕹️ Included Games

### 1. Memory Card Game
Test your memory by matching similar cards within minimum moves.

### 2. Typing Speed Test
Measure your typing speed, accuracy, and performance in real time.

### 3. Number Guessing Game
Guess the correct number with hints and limited attempts.

### 4. Word Scramble Game
Unscramble random words and improve vocabulary skills.
